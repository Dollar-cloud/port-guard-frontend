Port Guard is a UI/UX-focused frontend project that simulates an advanced Intrusion Detection System (IDS) dashboard for network security monitoring.

## Description
This project showcases a modern and interactive security dashboard experience, including a landing page, authentication flow, system boot loader, role-based access (user/admin), scope selection, and a real-time monitoring interface using mock data.

The primary focus of this project is **frontend architecture, UI/UX design, animations, and state-driven user interactions**.

## Features
- Animated landing page with dark/light mode
- System boot loader simulation
- User & Admin login UI
- Role-based interface flow
- Device scope selection (single, multiple, custom)
- Security dashboard with alerts & logs (mock data)
- Notification system UI
- Responsive and modern design
- Advanced animations and transitions

## Tech Stack
- HTML
- CSS
- JavaScript

## Notes
- This is a frontend-only project
- All alerts, logs, and system data are simulated (mock data)
- Backend functionality is not implemented
